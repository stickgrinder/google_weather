Google Weather by stickgrinder

Google Weather Drupal module.

This Drupal 6 module provide sort of a porting of the nice Joomla module RokWeather, implementing an sexy ajax-driven interactive block to show weather conditions and forcasts from Google Weather API.

Dependencies

Drupal 6.x

Install

Get the repo, place it somewhere inside your sites/all/modules folder (or its subfolders), enable the module in "Site building" -> "Modules" administrative section and place your new module where you want on the site. Remember to set default values, configuring module parameters in admin interface.
